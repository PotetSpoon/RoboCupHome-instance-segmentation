# RoboCupHome-instance-segmentation

The following is the code for image-pipeline and partial model training using dataset imported from Roboflow. Considering the training time amount, the model with all classes is trained locally. 

I'm responsible for computer vision of our model, which now only needs to fill the rest of classes up with more practical images. The robot is a humanoid which demands for the need to communicate, so I'm also responsible for the LLM implementation, which currently is considering of using OpenAI API.
